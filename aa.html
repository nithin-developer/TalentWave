We are seeking a highly skilled and passionate Full Stack Developer with strong expertise in the MERN (MongoDB, Express.js, React.js, Node.js) stack to join our engineering team. As a MERN Stack Developer, you will be responsible for designing, developing, and maintaining robust and scalable web applications, from the database layer to the user interface. You will play a crucial role in bringing our product visions to life, collaborating closely with product managers, designers, and other engineers to deliver exceptional user experiences and solid backend functionalities.

Key Responsibilities:

  • Full Stack Development: Develop and maintain both front-end and back-end components of our web applications using the MERN stack.
  • Frontend Development:
    • Build interactive, responsive, and high-performance user interfaces using React.js (including Hooks, Context API, and Redux for state management).
    • Ensure cross-browser compatibility and optimize applications for maximum speed and scalability.
    • Collaborate with UI/UX designers to translate wireframes and mockups into high-quality code.  
  • Backend Development:
    • Design and implement robust, scalable, and secure RESTful APIs using Node.js and Express.js.
    • Integrate third-party services and APIs as required.
    • Develop and manage database schemas and queries for MongoDB, ensuring efficient data storage and retrieval.
  • Database Management: Work with MongoDB to design and optimize database models, write efficient queries, and manage data migration.
  • Code Quality & Testing: Write clean, well-documented, and reusable code. Conduct unit, integration, and end-to-end testing to ensure application quality and reliability.
  • Collaboration: Participate in agile development methodologies (Scrum/Kanban), including daily stand-ups, sprint planning, and retrospectives.
  • Problem Solving: Troubleshoot and debug applications, identify bottlenecks, and implement solutions to improve performance and user experience.
  • Deployment & Maintenance: Assist with deployment processes, monitoring, and ongoing maintenance of applications.
  • Stay Updated: Keep abreast of new technologies and best practices in full-stack development, particularly within the MERN ecosystem.
